## ðŸš€ Getting Started

### Prerequisites

- Node.js & npm
- MongoDB (local or Atlas)
- Git

### Installation

```bash
# Clone the repo
git clone https://github.com/MalaikaTariq726/Mentorship-Platform-WebApp

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd client/app
npm install
```

### Run the Application

```bash
# Start backend server
npm run dev

# Start frontend client
npm start
```
